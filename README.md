# Калькулятор с историей

Простая консольная программа на Java.

## Возможности

- Сложение, вычитание, умножение, деление
- Сохранение истории операций
- Команда `history` — показать историю
- Команда `exit` — выход из программы

## Технологии

- Java 17
- Collections (ArrayList для истории)

## Как запустить

1. Скачайте файл `Calculator.java`
2. Откройте в IntelliJ IDEA
3. Запустите метод `main`

## Пример работы

=== Калькулятор с историей === 
Доступные операции: +, -, *, / 
Команды: 'history - показать историю, 'exit' - выход

Введите выражение (например, 2+2): 5+3 Результат: 8.0
Введите выражение (например, 2+2): history

=== История операций ===

1. 5+3 = 8.0
